# NewsReader
## 简介<br>
该项目主要用于毕业设计，界面设计了引导页，闪屏页，以及一个仿ZAKER的手推们的特效，基于Android 4.4开发，不对4.4以下版本进行适配。
数据从聚合数据处获得，通过RxVolley进行JSON数据的获取与处理，图片通过Picasso进行裁剪。<br>
![](https://github.com/taojinlei/NewsReader/blob/master/yanshi.gif)<br><br>
